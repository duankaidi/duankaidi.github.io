边分治神题。

前置知识：边分治，虚树。

题意

给定 333 棵有边权的树，每棵树都含有 nnn 个节点，令 disi(x,y)dis_i(x, y)disi​(x,y) 表示 (x,y)(x, y)(x,y) 在第 iii 棵树上的距离。求一组 (i,j)(i, j)(i,j)，使得 ∑k=13disk(i,j)\sum\limits_{k = 1}^3 dis_k(i, j)k=1∑3​disk​(i,j) 最大，为了方便，只需输出最大值。

题解

考虑边分治。下面用 TiT_iTi​ 来表示第 iii 棵树。

对 T1T_1T1​ 进行边分治，设 d1(x)d_1(x)d1​(x) 表示 xxx 点到分治边一侧的距离，那么答案就转化成了 d1(i)+d1(j)+dis2(i,j)+dis3(i,j)d_1(i) + d_1(j) + dis_2(i, j) + dis_3(i, j)d1​(i)+d1​(j)+dis2​(i,j)+dis3​(i,j)。然后将当前分治块内的所有点在 T2T_2T2​ 上拉出来，建一棵虚树 T2′T'_2T2′​，再设 d2(x)d_2(x)d2​(x) 表示 xxx 在 T2T_2T2​ 上的带权深度，答案又转化成了 d1(i)+d1(j)+d2(i)+d2(j)+dis3(i,j)−2d2pd_1(i) + d_1(j) + d_2(i) + d_2(j) + dis_3(i, j) - 2d_2{p}d1​(i)+d1​(j)+d2​(i)+d2​(j)+dis3​(i,j)−2d2​p，其中 ppp 表示 T2′T'_2T2′​ 中 (i,j)(i, j)(i,j) 的 LCA\text{LCA}LCA，显然 ppp 也是 T2T_2T2​ 中 (i,j)(i, j)(i,j) 的 LCA\text{LCA}LCA。

所以现在枚举 ppp，然后在 ppp 在 T2′T'_2T2′​ 的子树中，找到 (i,j)(i, j)(i,j) 使：1. i,ji, ji,j 在 T1T_1T1​ 的分治边的两侧。2. 答案最大。接着可以发现，上面的答案可以变成 (d1+d2)(i)+(d1+d2)(j)+dis3(i,j)−2d2(p)(d_1 + d_2)(i) + (d_1 + d_2)(j) + dis_3(i, j) - 2d_2(p)(d1​+d2​)(i)+(d1​+d2​)(j)+dis3​(i,j)−2d2​(p)，因为 ppp 是我们枚举的，故可以省略。接着，考虑把 (d1+d2)(i)+(d1+d2)(j)(d_1 + d_2)(i) + (d_1 + d_2)(j)(d1​+d2​)(i)+(d1​+d2​)(j) 给转化一下。一个精妙的处理方法是：在 T3T_3T3​ 中新建 i′,j′i', j'i′,j′，然后将 i′i'i′ 与 iii 之间连一条边权为 d1(i)+d2(i)d_1(i) + d_2(i)d1​(i)+d2​(i) 的边，对 jjj 和 j′j'j′ 同理。这样答案就转化成了 dis3(i′,j′)−2d2(p)dis_3(i', j') - 2d_2(p)dis3​(i′,j′)−2d2​(p)，也就是求 T3T_3T3​ 中的最长路（要保证 i,ji, ji,j 在分治边的两侧）。

我们发现，最长路竟然是可以合并的！也就是说：设 xxx 只有两个儿子，l,rl, rl,r 是 xxx 的左儿子子树的最长路两端点，l′,r′l', r'l′,r′ 是右儿子的。那么 xxx 的最长路两端点一定是 l,l′,r,r′l, l', r, r'l,l′,r,r′ 中的两个！根据这个结论，就可以在 T2′T'_2T2′​ 上面进行合并了。有一个细节，设 LLL 表示分治边左侧的点集，RRR 表示分治边右侧的点集，如果要保证 (i,j)(i, j)(i,j) 在分治边的两侧，那么先把 LLL 的最长路和 RRR 的最长路分别求出，然后再对两个最长路，分别选出一个点进行合并，这才能保证 i,ji, ji,j 在不同侧的限制。

于是这题就做完了，时间复杂度 O(nlog⁡2n)O(n \log^2 n)O(nlog2n)。
